# Strange checkers
Client-server GUI-app "Strange checkers" for playing checkers with a specific board. 
  * Client-server: Linux sockets

# Usage
## Client
make

./comp1 wind

./wind

## Server
./comp server

./server

Tested with 3 laptops, where 2 of them are clients and 1 is server. See comments on wind.h
